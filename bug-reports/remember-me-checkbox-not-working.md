# Bug Report: "Remember Me" Checkbox Does Not Keep User Logged In

## Bug ID
BUG-003

---

## Title
"Remember Me" functionality does not keep the user logged in after browser restart.

---

## Environment
- Browser: Chrome 136
- OS: Windows 11

---

## Preconditions
- User has a registered account
- User is on the login page

---

## Steps to Reproduce
1. Open the login page
2. Enter valid credentials
3. Enable the "Remember Me" checkbox
4. Click the "Login" button
5. Close the browser
6. Open the browser again
7. Reopen the website

---

## Expected Result
User should remain logged into the system.

---

## Actual Result
User is logged out and required to log in again.

---

## Severity
Medium

---

## Priority
Medium

---

## Status
Open
